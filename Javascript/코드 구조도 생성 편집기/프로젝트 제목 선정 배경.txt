네 가지 후보가 나왔다.


1. 구조도 기반 코드 분석기
2. 구조도 기반 코드 편집기
3. 구조도 방식 코드 편집기
4. 구조도 그려주는 코드 편집기


제목은 3으로 정했고,
발표할 때는 4로 소개한다.


4번 제목의 도출 배경은 3으로 발표하자니,
"구조도 방식" 이라는 프로젝트 고유 용어 대신에,
"구조도 그려주는" 이라는 직관적인 이름이 필요해 보였다.


그래서 프로젝트 제목을 4번으로 바꿀 생각도 해봤으나,
4번이 되면 "구조도 그려주는"이라는 표현이 마치,
"술파는 노래방" 처럼 구조도 그려주는 것이
부수적인 기능인 양 보여지기 때문에 제목은 유지하고,

대신 발표할 때만 직관적인 이름을 쓰는 것으로 선정했다.


코드 구조도 생성 편집기
